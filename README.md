## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install numpy and pygame.

```bash
pip install numpy
pip install pygame
```

## Features

1. It can generate any Julia Set with a complex number given (z = a + ib) and the Mandelbrot Set
